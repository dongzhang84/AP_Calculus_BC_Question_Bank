The definite integral \(\int_{0}^{4} \sqrt{x} \, dx\) is approximated by a left Riemann sum, a right Riemann sum, and a trapezoidal sum, each with 4 subintervals of equal width. If \(L\) is the value of the left Riemann sum, \(R\) is the value of the right Riemann sum, and \(T\) is the value of the trapezoidal sum, which of the following inequalities is true?

(A) \(L < \int_{0}^{4} \sqrt{x} \, dx < T < R\)
(B) \(L < T < \int_{0}^{4} \sqrt{x} \, dx < R\)
(C) \(R < \int_{0}^{4} \sqrt{x} \, dx < T < L\)
(D) \(R < T < \int_{0}^{4} \sqrt{x} \, dx < L\)
