Let \( g \) be a twice-differentiable, increasing function of \( t \). If \( g(0) = 20 \) and \( g(10) = 220 \), which of the following must be true on the interval \( 0 < t < 10 \)?

(A) \( g'(t) = 0 \text{ for some } t \text{ in the interval.} \)
(B) \( g'(t) = 20 \text{ for some } t \text{ in the interval.} \)
(C) \( g''(t) = 0 \text{ for some } t \text{ in the interval.} \)
(D) \( g''(t) > 0 \text{ for all } t \text{ in the interval.} \)
