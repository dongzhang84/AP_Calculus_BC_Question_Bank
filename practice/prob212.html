The function \(f\) has derivatives of all orders for all real numbers with \(f(0) = 3\), \(f'(0) = -4\), \(f''(0) = 2\), and \(f'''(0) = 1\). Let \(g\) be the function given by \(g(x) = \int_{0}^{x} f(t) \, dt\). What is the third-degree Taylor polynomial for \(g\) about \(x = 0\)?

(A) \(-4x + 2x^2 + \frac{1}{3}x^3\)
(B) \(-4x + x^2 + \frac{1}{6}x^3\)
(C) \(3x - 2x^2 + \frac{1}{3}x^3\)
(D) \(3x - 2x^2 + \frac{2}{3}x^3\)
(E) \(3 - 4x + x^2 + \frac{1}{6}x^3\)
